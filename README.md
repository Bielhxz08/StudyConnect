# 📚 StudyConnect

> Aplicativo mobile para conexão e formação de grupos de estudo entre estudantes.

---

## 📋 Sobre o Projeto

O **StudyConnect** é um aplicativo mobile desenvolvido em **React Native** como projeto acadêmico da disciplina de Desenvolvimento Mobile do curso de Análise e Desenvolvimento de Sistemas (ADS) — Estácio de Sá.

O app surgiu a partir de entrevistas com estudantes que relataram dificuldade em encontrar colegas para estudar juntos, organizar grupos por disciplina e manter comunicação eficiente entre os membros. O objetivo é facilitar a interação entre usuários por meio de grupos de estudo organizados, promovendo colaboração e engajamento acadêmico.

## 🖥️ Como Rodar o Projeto

### Pré-requisitos

- Node.js instalado
- Expo CLI instalado (`npm install -g expo-cli`)
- Aplicativo **Expo Go** no celular (Android ou iOS)

### Passos

```bash
# Clone o repositório
git clone https://github.com/SEU_USUARIO/studyconnect.git

# Entre na pasta do projeto
cd studyconnect/app

# Instale as dependências
npm install

# Inicie o projeto
npx expo start
```

Escaneie o QR Code com o aplicativo Expo Go para visualizar o app no celular.
